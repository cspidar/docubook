# Comenzar

## Introducción y cómo usar este documento

¡Hola! En este documento, te guiaremos a través de cómo aprender React.js fácilmente a través de ejemplos. React es una de las bibliotecas de frontend más utilizadas en el desarrollo web moderno, ayudándote a construir interfaces de usuario de manera eficiente. Este documento cubre React de manera sistemática, desde lo básico hasta temas avanzados, y está diseñado para ser útil cuando trabajes en proyectos reales.

### Usarlo como punto de partida y base

Este documento está escrito para aquellos que son nuevos en React. Comienza con los conceptos básicos y la configuración de React, avanzando gradualmente hacia conceptos más complejos, permitiéndote aprender a través de ejemplos prácticos. Cada capítulo explica conceptos clave e incluye ejemplos prácticos para ayudarte a comprender escribiendo tu propio código.

### Aprendizaje avanzado y exploración adicional

Cada capítulo distingue entre conceptos esenciales que necesitas conocer y temas avanzados opcionales para un estudio más profundo. Los conceptos avanzados que no se cubren en el texto principal no son esenciales en el momento del aprendizaje, pero se pueden consultar de manera selectiva cuando sea necesario. Por lo tanto, después de dominar los conceptos básicos, puedes profundizar en el conocimiento mediante materiales de aprendizaje adicionales.

### Cómo navegar por el documento

Cuando desees volver a esta página, simplemente haz clic en el título en la parte superior izquierda. También puedes abrir la pestaña de la izquierda para ver el menú completo y navegar a la sección deseada. El último capítulo proporciona enlaces relacionados para un aprendizaje avanzado, así que asegúrate de revisarlos.
