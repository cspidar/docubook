# 시작하기

다양한 지식을 간편하게 확인할 수 있는 무료 전자책을 제공합니다. 아래 내용을 참고하세요.

## 검색

오른쪽 위의 <i class="fa-solid fa-magnifying-glass"></i> 메뉴에서 원하는 주제를 검색할 수 있습니다.

## 탐색

왼쪽 메뉴에서 원하는 주제를 선택하세요. 만약 보이지 않으면, 왼쪽 위의 <i className="fa-solid fa-bars"></i> 버튼을 선택해 메뉴를 열 수 있습니다.

이 페이지는 언제든 왼쪽 위의 `무료 전자책` 문구를 선택해 돌아올 수 있습니다.
